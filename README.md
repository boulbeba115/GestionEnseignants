# Teachers Management App
Teacher Management Application made with (JEE, Servlet, JSP) 
# Feature list 
- Simple static authentication (based on session)
- Teachers Crud
- Class Crud
- Subject matter 
- Grade Crud
- Level Crud
- Department Crud

- Netbeans or Eclipse Jee 
- web server (GlassFish or tomcat)
